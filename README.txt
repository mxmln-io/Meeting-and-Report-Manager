-------------------------------------------------------------------------------
                              CLIENT SCHEDULING APPLICATION
-------------------------------------------------------------------------------

This application is designed to manage client schedules, track user activity, and generate various reports for administrative purposes.

-------------------------------------------------------------------------------


Development Environment:

IDE:
IntelliJ Community [2023.1.3]

JDK:
Java SE [20.0.1]

JavaFX:
JavaFX-SDK-[20]

-------------------------------------------------------------------------------

Directions to Run the Program:
1. Open the application using IntelliJ IDE.
2. Ensure that the necessary libraries, especially JavaFX and MySQL Connector, are included in the project structure.
3. Navigate to the 'Main' class and click on the 'Run' button to start the application.
4. Follow on-screen prompts and use the provided login credentials to access the dashboard.

-------------------------------------------------------------------------------

Additional Report (Part A3f):
The additional report generated provides insights into ["The total number of appointments by customer"]

-------------------------------------------------------------------------------

MySQL Connector Version:
mysql-connector-java-[8.1.0]
